InfiniteHomes Minecraft Plugin

Unbegrenzte oder kontrollierbare Homes für deine Spieler! Leichtgewichtig, einfach zu bedienen und perfekt für jeden Survival- oder Freebuild-Server.

Jetzt herunterladen

Plugin-Features

Unbegrenzte Homes

Spieler können standardmäßig so viele Homes setzen, wie sie möchten - oder du als Serverbetreiber legst fest, wie viele Homes erlaubt sind.

Einfache Konfiguration

Keine Datenbank erforderlich! Einfach installieren und loslegen. Perfekt für kleine und große Server.

Multilingual Support

Unterstützung für mehrere Sprachen durch benutzerdefinierte Textdateien. Einfache Anpassung aller Plugin-Nachrichten.

Cooldown-System

Konfigurierbare Cooldowns zwischen Teleportationen, um Missbrauch zu verhindern.

Leistungsoptimiert

Leichtgewichtiges Plugin, das keine Serverressourcen überlastet und selbst auf großen Servern reibungslos läuft.

Permissions

Vollständige Integration mit Permission-Systemen, um differentierte Zugriffsrechte zu verwalten.

Befehle

/sethome <name>
Setzt ein Home mit einem benutzerdefinierten Namen
/home <name>
Teleportiert dich zu deinem Home
/delhome <name>
Löscht ein Home
/homes
Listet alle Homes des aktuellen Benutzers auf
/homecooldown <number/-1>
Setzt den Cooldown zwischen Teleports (OP only, -1 = kein Cooldown)
/homecount <number/-1>
Setzt die maximale Anzahl von Homes (OP only, -1 = unbegrenzt)

Installationsanleitung

  1. Lade die neueste Version des Plugins von einer der Plattformen herunter
  2. Füge die JAR-Datei in den Plugins-Ordner deines Minecraft-Servers ein
  3. Starte den Server neu, um das Plugin zu laden
  4. Passe die Konfiguration in der generierten Config-Datei nach Bedarf an
  5. Füge benutzerdefinierte Übersetzungen im /plugins/Infinitehomes/translations/ Ordner hinzu

Häufig gestellte Fragen

Wie installiere ich das InfiniteHomes Plugin?
Lade die JAR-Datei herunter, lege sie in den Plugins-Ordner deines Servers und starte den Server neu. Das Plugin generiert automatisch eine Config-Datei, die du nach deinen Wünschen anpassen kannst.
Unterstützt das Plugin mehrere Sprachen?
Ja, das Plugin unterstützt mehrere Sprachen. Du kannst benutzerdefinierte Übersetzungen im /plugins/Infinitehomes/translations/ Ordner hinzufügen, z.B. texts_de.yml für Deutsch.
Kann ich die maximale Anzahl von Homes pro Spieler begrenzen?
Ja, als Serveroperator kannst du mit /homecount <number> die maximale Anzahl von Homes festlegen. Verwende -1 für unbegrenzte Homes.
Ist eine Datenbank für dieses Plugin erforderlich?
Nein, das Plugin ist leichtgewichtig und benötigt keine Datenbank. Alle Daten werden in lokalen Dateien gespeichert.
Kann ich Cooldowns für Teleports festlegen?
Ja, du kannst mit /homecooldown <seconds> einen Cooldown zwischen Teleportationen festlegen. Verwende -1, um Cooldowns zu deaktivieren.
